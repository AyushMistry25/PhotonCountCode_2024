To visually inspect your color channel of interest within the defined region, and also plot a photon count plot as a function of time, we run the following code chunk. The force plot is not of immediate use, but can help detect any disturbance on the beads that might have occured due to collision with debris in the microfluidic channel(s). You can also export the data points for the photon count plot as a .csv file. The last two lines of the code chunk can be used to save the data for photon counts as a .csv file and save the combined figure as a .png file.
